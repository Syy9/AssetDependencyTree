# AssetDependencyTree
Unity Asset dependency tree viewer

![demo](demo1.gif)


## Feature
* Show Asset Dependencies in TreeView
* Filter Asset Dependencies

## Usage
1. Select target assets.
2. Click MenuItem "Select Dependencies(Tree View)"
